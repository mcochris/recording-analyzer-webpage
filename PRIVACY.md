# Privacy Policy

**Last updated:** April 2026

## Overview

The recording-analyzer website can be used display JSON data generated by the recording analyzer
application found at <https://github.com/mcochris/Recording-analyzer>.

## What the website does

- Reads a JSON file that you explicitly upload to the website
- Analyzes the JSON data and displays results to your terminal
- Collects log data, including IP addresses, browser types, and pages visited

## What the website does NOT do

- It does not collect personal data
- It does not display advertising
- It does not read files beyond those you explicitly provide
- Does not use cookies
- Does not track you

## Open source

The complete source code is available on GitHub. You are encouraged to review it and
verify these claims for yourself.

<https://github.com/mcochris/recording-analyzer-webpage>

If you have questions about this policy, feel free to open a GitHub issue at <https://github.com/mcochris/recording-analyzer-webpage/issues>
