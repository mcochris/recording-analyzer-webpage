# README

![License](https://img.shields.io/badge/License-MIT-yellow)

This website is associated the recording-analyzer tool, a tool for analyzing objective characteristics of audio files.
Source code for the recording-analyzer tool can be found at <https://github.com/mcochris/Recording-analyzer>

recording-analyzer has an option to output the statistics it generates to the JSON data format. The user can upload
the JSON file to this website via the "Load JSON file" button. This website analyzes the data and displays it in a
tabulator format.

## Features

- Each column of the table can be sorted by clicking the column header.
- Columns can be moved by clicking and dragging the column to a new position.
- Rows can be moved by clicking and dragging the three line icon in the tables first column to a new position.
- Rows can be selected by clicking them.
- Adjacent rows can be selected by clicking on the first row you want selected, holding down the shift button on the keyboard,
then clicking on the last row you want selected.
